## <a href="https://www.audacityteam.org/" >Audacity</a>
Audacity is a popular free software for recording and editing audio.

## [Persepolis Download Manager](https://persepolisdm.github.io/)
This third-party application may provide a solution if you have problems downloading our content. You can use any other download manager. This one has the advantage of being open-source.

## <a href="https://www.screentogif.com/" >Screen-to-gif</a>
Handy screen capture for video with a simple in-built editing suite. Can save in `webm`, `gif` and many more formats.

## <a href="https://getsharex.com/" >Share-X</a>
This screen video capture tool also lets you record or add sound to the video

## <a href="https://code.visualstudio.com/" >Visual Studio Code (VSCode)</a>
Very useful and powerful text and code editor. Highly recommended.

## <a href="https://support.microsoft.com/en-us/windows/use-snipping-tool-to-capture-screenshots-00246869-1843-655f-f220-97299b865f6b#:~:text=Press%20Windows%20logo%20key%20%2B%20Shift%20%2B%20S.,Press%20Print%20Screen%20(PrtSc)." >Windows Capture Tools</a>
Press `ALT + SHIFT + S` if you're on windows to open the Snipping tool. It lets you capture a custom screenshot very quickly and you can add simple annotations to it if you need. We use it all the time!
