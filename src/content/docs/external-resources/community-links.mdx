## Foundry VTT
<Card title="Virtual Tabletop" icon="open-book">
	<p>If you didn't already know, this is the virtual tabletop we use. 
	We found it very functional and inexpensive for how awesome and flexible it is, but it does 	require a bit of computer knowledge to be used correctly. However, it's very well documented, 	has lots of game systems, can be customised thanks to modules and a very friendly and active 	Discord server.</p>
	<a href="https://foundryvtt.com" target="_blank">Official Website</a>
	<p></p>
	<a href="https://foundryvtt.com/kb/" target="_blank">Documentation</a>
	<p></p>
	<a href="https://discord.com/invite/foundryvtt" target="_blank">Discord Server</a>
	<p></p>
	<a href="https://www.patreon.com/foundryvtt" target="_blank">Patreon</a>
</Card>
